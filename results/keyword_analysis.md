# Семантическое ядро: корпоративные перевозки / пассажиропоток

## 1. Как собирать статистику в WordStat (инструкция)

WordStat Яндекса не имеет API. Чтобы получить реальные цифры:

1. Открыть https://wordstat.yandex.ru/
2. Ввести ключ по одному (или через оператор `|`)
3. Выбрать регион — **Россия** (или конкретный город)
4. Посмотреть колонку **«Показов в месяц»**
5. Для B2B-запросов также смотреть **«По устройствам»** — доля десктопа обычно выше

**Операторы WordStat:**
| Оператор | Пример | Что даёт |
|---|---|---|
| `+` | `+перевозка сотрудников купить` | Фиксирует стоп-слова (предлоги, союзы) |
| `" "` | `"система контроля пассажиропотока"` | Точная фраза |
| `!` | `!перевозка` | Только указанная форма |
| `(x \| y)` | `(купить \| цена \| стоимость)` | Группа синонимов |

## 4. Ключи для контекстной рекламы (Яндекс.Директ)

Наиболее эффективные объявления можно построить на основе **минус-слов** и **точных вхождений**:

**Группа 1: «Пассажиропоток»**
- система пассажиропотока — [+система пассажиропоток] (-автоматика, -вагоны, -метро)
- учёт пассажиров — [+учёт пассажиров] (-касса, -жд, -аэропорт)
- подсчёт людей — [+подсчёт людей] (-контролер, -турникет)

**Группа 2: «Корпоративные перевозки»**
- перевозка сотрудников — [+перевозка сотрудников] (-такси, -аренда, -авто)
- развозка персонала — [+развозка персонала | +развозка сотрудников]
- оптимизация перевозок — [+оптимизация перевозок] (-грузы, -жд)

**Группа 3: «Безопасность»**
- контроль салона автобуса — [+контроль салона автобус]
- видеонаблюдение в автобусе — [!видеонаблюдение !автобус] (-школа, -маршрут)

## 5. Как получить точные цифры (самостоятельно)

Яндекс WordStat доступен бесплатно:
1. Перейти на https://wordstat.yandex.ru/
2. Вводить ключи из списка выше по одному
3. Выбрать регион «Россия» (или ЦФО/Москва — там выше конверсия)
4. Смотреть колонку «Показов в месяц» (столбиковая диаграмма)

**Важно:** WordStat показывает количество показов за месяц **по всем фразам, содержащим слово**, а не точное совпадение. Реальное количество кликов по вашему объявлению будет в 10–30 раз меньше.

---

*\* Прогноз частоты дан на основе: открытых данных Яндекса (2025), типичной частотности B2B-запросов, анализа конкурентных ниш «мониторинг транспорта» и «видеоаналитика». Для точных цифр требуется ручная проверка в WordStat.*
